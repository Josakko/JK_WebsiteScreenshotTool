About JK Website Screenshot Tool

This is a website screenshot tool created by Josakko that allows you to screenshot any website whit just its URL and save the screenshot as a image file. Visit my github here.

To use this tool, all you need to do is paste the URL of the website that you want to screenshot and click on "Generate Screenshot" button and wait until the capturing screenshot is done! If you need any help or find any bugs, please contact me on my discord server.
